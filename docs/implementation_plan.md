# 液态折射实现计划

## 目标范围

仅实现液态折射相关能力：

- 位移场更新
- 折射采样
- 高光与金属感
- 可选色散
- 图片与文字两类输入

不纳入当前仓库范围的内容：

- 粒子系统
- 页面导航组件
- 通用背景容器
- 与业务页面强绑定的演示逻辑

## 参考实现拆解

参考网页效果可以拆成四层：

1. 底图输入
2. 液面位移场
3. 法线推导与折射采样
4. 色散和高光增强

## Flutter 迁移建议

### 第一阶段

- 已完成：手势扰动注入、连续位移场、位移纹理编码、shader 折射层和示例页
- 当前实现采用“连续位移场 + fragment shader 折射采样”
- 这一阶段的目标是先把交互模型、参数边界和移动端渲染链路打通

### 第二阶段

- 把文字舞台和图片输入继续拆分成更清晰的接入方式
- 优化位移纹理更新频率和 shader 参数组织
- 继续加强连续位移场的采样稳定性和高光表现

### 第三阶段

- 强化法线、高光和材质质感
- 继续补细粒度参数与性能策略
- 视需要补运行时能力检测和更细的错误提示

## 结构建议

- `LiquidRefractionSurface`：对外组件入口
- `LiquidRefractionConfig`：对外参数对象
- 后续新增的位移场、shader 和交互控制器都放进 `lib/src/` 下，不暴露多余实现细节
